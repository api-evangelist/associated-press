# Associated Press (associated-press)
The Associated Press (AP) is an American not-for-profit news agency. The AP provides APIs for accessing election data, news content, and media assets. The AP Elections API enables integration with election systems to retrieve election race information and results for powering election websites and applications.

## APIs

### AP Elections API
Integrate your election systems with AP Elections API. Your election results delivery application retrieves election race information from AP Elections API to power election websites and reporting systems.
